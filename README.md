#ИГРА КРЕСТИКИ-НОЛИКИ (Tic-tac-toe) для произвольной доски

#Особенности

Есть 2 варианта реализации:

1) с применением angular.js
2) без применения angular.js (чистый java script + canvas)


запуск приложения

1) npm install http-server -g
2) npm install bower -g
2) bower install
3) http-server ./app
4) открыть в браузере страницу localhost:808x (где x - номер порта при запуске)
5) есть вариант приложения с использованием чистого js + canvas (страница index_vanilla.html)
